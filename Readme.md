# SafeAround

SafeAround is a community-driven personal safety mobile application designed to help users make safer movement decisions and receive immediate assistance during emergencies.

The app focuses on safety-aware routing, area safety insights, and fast SOS broadcasting to nearby users and authorities.

---

## Key Features (MVP)

### SOS Emergency System
- Send SOS alerts to:
  - Nearest police station
  - Emergency contacts
  - SafeAround users within a 100m radius
- Nearby users receive the alert without live location sharing
- Designed to minimize response time while preserving user privacy

### Area Safety Insights
- View safety heatmaps of areas before visiting
- Users can rate areas based on their experience
- Community data contributes to safety intelligence

### Safe Routing (Planned)
- Provides alternate routes if the shortest route is unsafe
- Route safety scoring based on historical and community data
